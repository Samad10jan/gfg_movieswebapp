# 🎬 GFG Movies Web App

A React practice project built to learn and implement **Redux Toolkit**, **Redux Thunk**, and modern React development using **TypeScript** and **Vite**.

## 🚀 Overview

This project is a movie browsing application where users can search for movies, view details, and experience asynchronous state management using Redux Toolkit and Thunk.

The primary goal of this project was to gain hands-on experience with:

- Redux Toolkit
- Redux Thunk
- Async API handling
- Global state management
- React Router navigation
- TypeScript in React applications

## ✨ Features

### Movie Browsing
- Search movies by title
- Browse movie listings
- View detailed movie information

### State Management
- Global state management using Redux Toolkit
- Organized Redux slices
- Async actions handled with Redux Thunk

### API Integration
- Fetch movie data from external APIs
- Loading states during requests
- Error handling for failed requests

### User Experience
- Responsive UI
- Dynamic routing with React Router
- Material UI components with Tailwind CSS styling

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd gfg-movies-web-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the URL shown in the terminal.

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
